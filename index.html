Counter.nml 
state count = 0

div.container {
  style {
    text_align: "center"
    padding: "2rem"
  }

  h1 "Count: " count
  
  button(on: click { count.value += 1 }) {
    style {
      background_color: "#42b983"
      color: "white"
    }
    "Increment"
  }
}

Philosophy

開発者が「書く」ことに
集中できる言語

NMLは、Web開発の複雑さを取り除き、純粋な創造性を取り戻すために設計されました。

統合構文 (Unified Syntax)

HTML、CSS、JavaScript を1つのファイル、1つの構文に統合。冗長なタグやコンテキストスイッチはもう必要ありません。Rubyのように直感的でエレガンスです。

ビルトイン・リアクティビティ

state 宣言をするだけで、NMLは変数の変更を追跡し、自動的にUIを更新します。複雑なDOM操作や仮想DOMのオーバーヘッドを意識する必要はありません。

スコープスタイル

コンポーネント内で定義されたスタイルは、自動的にそのコンポーネント内にカプセル化されます。クラス名の衝突や、CSS設計の悪夢は過去のものです。

数秒で始められます

NMLのCLIツールは軽量で、設定不要です。npm経由でインストールし、すぐにコーディングを開始できます。

$ npm i nml-compiler

Next Steps:

  • 1
    プロジェクトを作成: nml create my-app
  • 2
    開発サーバーを起動: cd my-app && nml dev
  • 3
    ビルド: nml build
