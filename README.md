
# NML (Noppo Markup Language)

This project is a simple, multi-page static website for the fictional Noppo Markup Language (NML). It is built with vanilla HTML and JavaScript, using Tailwind CSS for styling.

## Pages

- index.html: The main landing page.
- document.html: The documentation page.
- playground.html: An interactive playground to try out NML.

## Scripts

- main.js: Handles common UI interactions like the mobile menu and header effects.
- playground-logic.js: Contains the logic for the NML playground, including a mock compiler.
- chat-widget.js: Powers the AI chat assistant.

## Getting Started

1.  Open any of the .html files in your web browser to view the pages.
2.  To enable the AI Chat Assistant, you must add your Google AI API key to the API_KEY constant in chat-widget.js.
